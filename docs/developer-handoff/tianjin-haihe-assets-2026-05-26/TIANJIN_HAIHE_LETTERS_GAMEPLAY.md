# 天津《海河来信》玩法与配套 PNG 清单

版本：v0.1 / 可直接给技术与视觉继续接入

## 章节定位

《海河来信》把天津设计成一条可换岸、可回信、可听声的城市路线。北京章节以“中轴秩序”作为骨架，天津章节更适合用“河流通信”作为骨架：玩家沿海河与周边街区行走，照片、声音、桥梁、街区立面和民俗手艺都会变成一封封信，最终寄回海河。

推荐主线：世纪钟 / 海河起点 -> 解放桥 -> 解放北路 -> 古文化街 -> 意风区 -> 五大道或海河夜景终局。

## 可行性玩法

- 来信机制：每个节点不是单纯打卡，而是收到一个历史角色的问题，玩家用照片、声音或手选元素回信。
- 换岸机制：桥、船票、转弯事件让路线可以根据体力、天气、人流动态切换，不强迫完整走完。
- 立面阅读：天津非常适合“窗、砖、门、柱、阳台”的局部观察，能降低历史知识门槛。
- 市声采样：河岸水声、车铃、人声、茶馆声、远处机械声都能成为低成本交互。
- 津味手艺：年画、泥人、茶馆、点心和街市可以承担轻松的亲子与生活线。
- 夜航终局：天津夜景强，适合把结尾设计成灯影、桥影和回信牌阵。

## 终局牌阵

终局建议按五栏展示：

1. 我从哪里收到信：海河来信卡、第一封河信、钟针线索卡。
2. 我怎样过河：桥影卡、桥上见证、换岸船票卡、守桥人。
3. 我读到哪面墙：万国窗卡、砖窗卡、立面读者、洋行账房。
4. 我听见什么：河声卡、船笛卡、市声事件卡、茶馆说书人。
5. 我寄回什么：回声事件卡、回信明信片、夜航灯卡、海河点灯人。
