# Tianjin Haihe developer handoff

This folder is an additive handoff package for the Tianjin "Haihe Letters" chapter.
It does not replace or modify the existing app assets.

## Contents

- `TIANJIN_HAIHE_LETTERS_GAMEPLAY.md`: gameplay and asset mapping notes.
- `metadata/manifest.json`: full file manifest with dimensions, modes, and source notes.
- `metadata/PACKAGE_README.md`: README from the generated asset package.
- `zips/tianjin-haihe-cards-01-20.zip`: card PNG assets 01-20.
- `zips/tianjin-haihe-cards-21-40.zip`: card PNG assets 21-40.
- `zips/tianjin-haihe-tile-scenes-01-24.zip`: tile scene JPG assets 01-24.

## Asset Counts

- Cards: 40 transparent PNG files.
- Tile scenes: 24 compressed JPG files.
- Each zip also includes the gameplay Markdown and manifest for context.
